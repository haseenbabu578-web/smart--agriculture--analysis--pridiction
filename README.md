# Crop Prediction Web App 🌱

## Overview

This project is a **Flask-based web application** that predicts crop-related results based on environmental and soil parameters.
The application takes user input such as **Crop ID, Soil Type, Seedling count, Moisture, Temperature, and Humidity**, processes it using a trained **machine learning model**, and returns a prediction result.

This project demonstrates how to **deploy a machine learning model using Flask** and build a simple **web interface for predictions**.


## Features

* Web interface for entering crop and environmental data
* Machine learning model integration
* Automatic feature encoding using `pandas.get_dummies()`
* Real-time prediction output
* Easy to run locally



## Technologies Used

* **Python**
* **Flask**
* **Pandas**
* **Joblib**
